## Method

Exact rational pmf first, Monte-Carlo agreement and structural invariance second.

**Exact pmf backbone (`fractions.Fraction`, no floats).** `exact_pmf_sequence(nmax)` builds the full pmf of `C_n` for n = 0..nmax by rational dynamic programming: `P_0 = {0: 1}` and the transition `P_{k+1}(c) = P_k(c)·(1 − 2^{−c}) + P_k(c−1)·2^{−(c−1)}` — from state `c` you increment (to `c+1`) with probability `2^{−c}` and stay with probability `1 − 2^{−c}` (at `c=0`, `2^0 = 1`, so the first increment always fires). `moment_2c` and `moment_4c` read off `E[2^C] = Σ 2^c P(c)` and `E[4^C] = Σ 4^c P(c)` as exact Fractions, from which `E[N̂] = E[2^C] − 1` and `Var[N̂] = E[4^C] − (E[2^C])^2`. G1 asserts, for every n = 0..80 and with zero tolerance, `E[2^C] == n+1`, `E[N̂] == n`, `E[4^C] == 1 + 3n(n+1)/2`, `Var[N̂] == Fraction(n(n−1), 2)`, plus that the pmf sums to exactly 1 (a fifth independent per-n check) — `checked = 405 = 81×5`, `mismatches = 0`.

**Monte-Carlo (genuinely iid).** `morris_batch(n_inc, replicas)` re-seeds `random.seed(SEED)` at the START, then runs `replicas` independent Morris runs of `n_inc` increments each (`if random.random() < 2.0**(−c): c += 1`), forming `est = 2^C − 1` per replica. The sufficient statistics — Σ`est`, Σ`est²`, Σ`C`, Σ`C²` — are accumulated as **exact Python integers**, so no float summation-order instability enters the payload; the mean, honest iid SE (`sample_sd/√M`), and `z` are computed from those exact sums. Each replica is independent, so the plain iid SE is the honest one (no batch means / thinning). G2 runs the headline scale n=63, M=300000 and **stores** the batch so G4 reuses the SAME sample without re-simulating.

**Structural invariance + second scale.** G3(a) reads the exact `E[2^{C_n}]` list off the G1 pmfs and checks the **martingale increment** `E[2^{C_{n+1}}] − E[2^{C_n}] == 1` for n = 0..79 — this exact +1 step is the structural reason `E[2^{C_n}] = n+1` at every `n`. G3(b) re-seeds and runs a second, DIFFERENT scale n=31, M=300000 to confirm unbiasedness is not an artifact of the n=63 choice.
