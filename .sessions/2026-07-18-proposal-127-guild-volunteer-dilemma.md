## 💡 Session idea
"Every additional independent decider whose contribution is a SUBSTITUTE for everyone else's dilutes the equilibrium probability that ANYONE acts" is a reusable diffusion-of-responsibility audit lens for any collective-action surface. The volunteer's dilemma is the pure case: when a public good needs only ONE contributor, each rational member free-rides harder as the group grows, so the probability nobody acts RISES with N (toward c/b) while the expected total contribution stays ~constant (→ −ln(c/b)) — adding members buys dilution, not effort. The transferable corrections mirror the double-marginalization lens from P126 but for the provision side: (1) count the number of members whose action is a perfect SUBSTITUTE for the others' (the bystander set), not the raw roster — a bigger substitute set lowers per-member responsibility multiplicatively; (2) the cure is never "recruit more members" but "remove the substitutability" — ASSIGN the duty (a named tank/soaker/officer turns the mixed game into a dominant-strategy pure one), split the reward so a contributor keeps a private surplus (b−c internalized, the two-part-tariff analogue), or add a small volunteer bonus that makes volunteering dominant; (3) any place where one actor suffices and the rest can free-ride is a candidate — raid mechanic soaks, guild-bank restocking, event staffing, open-source maintenance, on-call rotations, community moderation. Candidate future game/mechanism heads with the same "substitutable contribution dilutes provision" spine: the THRESHOLD volunteer's dilemma (k-of-N must contribute — the failure law and the optimal squad size shift), the assigned-duty cure quantified (a named role restoring provision to ~1), and the reward-share fix (a contributor kickback that makes volunteering dominant and provision certain).
