## What happened
- Idea: a fleet of N agents each estimates a common truth θ (WLOG θ=0); agent i's error is ε_i=√ρ·C+√(1−ρ)·Z_i — a SHARED common-mode draw C~N(0,σ²) (identical for every agent: same config, upstream, prior, or dataset) plus an idiosyncratic Z_i~N(0,σ²) iid — so every agent's error has variance σ² and every pair has correlation ρ. The naive "wisdom of crowds" reading says averaging N estimates cuts the error variance to σ²/N→0. It does not: the common-mode term is the same in every estimate and does not average away, so the fleet-mean error variance is the exact equicorrelation identity MSE_N(ρ)=ρσ²+(1−ρ)σ²/N, which FLOORS at ρσ² as N→∞, and the effective independent count N_eff=σ²/MSE saturates at 1/ρ. ## 💡 Session idea
"A fleet with residual pairwise error-correlation ρ has an aggregate MSE floored at ρσ² and an effective independent count N_eff=1/[ρ+(1−ρ)/N]→1/ρ" is a reusable precision-budgeting lens for any fleet that averages or votes over many agents to beat noise. Whenever many replicas/monitors/models/pollers are pooled but share an error source — the same stale cache, the same upstream, the same training prior or feature, the same failing dependency, the same clock — the shared component never averages away, so the count N is a mirage and the real precision is N_eff=1/ρ. The transferable audit: measure the residual correlation ρ, report N_eff (not N) as the precision budget, decorrelate the shared input/config/prior before scaling the count, and stop adding agents past N≈1/ρ because the count grows while the precision does not. Candidate future fleet/estimation heads: quorum monitors on one upstream reading "5-nines" that is really 1/ρ-nines; model ensembles sharing a feature or pretraining corpus whose ensemble accuracy saturates far below the independent-Condorcet limit; retry fan-out against a correlated-failure backend; and any "just add more workers/replicas/voters" scaling decision priced on N instead of N_eff — each a "shared-common-mode ⇒ MSE floor ρσ², N_eff→1/ρ" claim with the same exact equicorrelation-variance anchor and a ρ=0 no-floor control σ.
