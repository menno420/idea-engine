## What this proposal does
Adds a superbot-games PROPOSAL establishing the odious closed form of Mock Turtles, an impartial coin-turning game (normal play, last to move wins). A move turns over 1, 2, or 3 coins with the rightmost turned coin going Heads→Tails (so play always terminates). The Sprague–Grundy value of a single heads-up coin at position n is the Mock-Turtle number g(n) = the unique odious member of {2n, 2n+1} = 2n if popcount(n) is odd else 2n+1 — the odious numbers 1,2,4,7,8,11,13,14,16,… (OEIS A000069). By the coin-turning decomposition (a corollary of the Sprague–Grundy sum theorem) the value of any position with heads at set S is the nim-sum ⊕ₙ∈S g(n), so the first player wins iff that nim-sum is nonzero. Ships a stdlib-only firsthand verifier that pits an independent from-scratch mex-DP game oracle (knows nothing of the closed form) against the odious closed form through four gates each in its own direction. ## Method
Ground-truth engine first, closed form second, then the two are forced to agree. (1) The game oracle grundy(pos) computes the Sprague–Grundy value by memoized mex-DP over frozenset positions: a legal move picks a head h (the rightmost turned coin, H→T) plus any 0/1/2 extra indices below h to toggle, giving successor pos△({h}∪extra); the value is the mex of successor values, grundy(∅)=0 — the oracle plays the game, it assumes no theorem.
